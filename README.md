## ✨ Özellikler

- 🏪 **AI App Marketplace** — AI uygulamalarını keşfet ve yönet
- ⚙️ **Uygulama Yönetimi** — Yapılandırma ve izleme paneli
- 👥 **Kullanıcı & Rol Yönetimi** — Admin, kullanıcı rolleri
- 🌐 **Website Builder** — Sürükle-bırak site oluşturucu
- 🔐 **Authentication** — Güvenli giriş sistemi
- 🎨 **Modern UI** — Gradient tasarım, Heroicons

## 📁 Proje Yapısı

```
src/
├── views/              # 9 sayfa (Home, Apps, Login, vb.)
├── components/         # 10 bileşen klasörü
├── stores/             # Pinia state yönetimi
├── services/           # API servisleri
├── router/             # Vue Router
├── types/              # TypeScript tanımları
└── assets/             # Görseller ve stiller
```

## 🚀 Kurulum

```bash
git clone https://github.com/mvtandas/papyon-ai-frontend.git
cd papyon-ai-frontend
npm install
npm run dev
```

> ⚙️ Backend için: [papyon-ai-backend](https://github.com/mvtandas/papyon-ai-backend)
